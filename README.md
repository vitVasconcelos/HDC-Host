# 🌐 HDC Host — Landing Page Responsiva

Este projeto é uma **landing page completa e responsiva** para uma empresa fictícia de hospedagem chamada **HDC Host**.  
A página apresenta serviços, planos, formulário de contato e uma área para pesquisa de domínio, tudo estruturado com **HTML5** e estilizado com **CSS3**.
