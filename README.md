# 🌐 HDC Host — Landing Page Responsiva
Este projeto é uma **landing page completa e responsiva** para uma empresa fictícia de hospedagem chamada **HDC Host**.

A página apresenta serviços, planos, formulário de contato e uma área para pesquisa de domínio, tudo estruturado com **HTML5** e estilizado com **CSS3**. Também foi utilizado **JavaScript** para adicionar interatividade e melhorar a experiência do usuário.

Entre as funcionalidades implementadas, destaca-se o **menu de navegação fixo**, que permanece no topo da tela e direciona os usuários para as principais seções do site. Durante a rolagem da página, o link correspondente à seção visível recebe destaque automático, facilitando a navegação.

O formulário de contato conta com **validação dinâmica dos campos**, verificando nome, e-mail e mensagem. Mensagens de erro são exibidas quando os dados estão incompletos ou inválidos, e, após o envio correto, os campos são limpos automaticamente e o usuário recebe uma confirmação de sucesso.

O projeto demonstra a aplicação prática de conceitos de **estrutura semântica, responsividade, manipulação do DOM, eventos e validação de formulários**, sendo uma ótima representação de habilidades em desenvolvimento frontend.
